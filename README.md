# Portal HIV/AIDS — Informação, Prevenção e Acolhimento

Projeto de extensão universitária desenvolvido na disciplina de Inteligência Artificial para Devs.

## O que é

Portal web informativo sobre HIV/AIDS com assistente virtual de IA (Ana), baseado nas diretrizes oficiais do Ministério da Saúde do Brasil.

## Tecnologias

- Java 17 + Spring Boot 3.2.5
- MySQL 8
- Thymeleaf (frontend)
- API Anthropic Claude (IA do chatbot)
- Hospedagem: Railway
